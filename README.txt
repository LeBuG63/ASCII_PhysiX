Controles:
[h]	L'aide
[l] Poser un bloc léger
[L] Poser un bloc lourd
[r] Poser un bloc soumis à la gravité inverse
[ ] Poser un mur

[ENTREE] Mettre en pause

-----------------------------------------------------------

Options:
Vous pouvez modifier les options dans le fichier "options.opt" /!\ NE SOURTOUT PAS L'EFFACER !
"widht=NN" 		La largeur de la map (ne pas dépasser 79)
"height=NN" 	La hauteur de la map
"char_border=C"	Le caractère pour la bordure de la map
"char_fill=C"	Le caractère pour l'intérieur de la map

N représente un nombre
C représente un caractère
